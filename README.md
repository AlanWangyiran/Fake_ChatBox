# Fake_ChatBox 模拟聊天框

> tkinter版本：tk == 8.6.12 

## 使用方法

运行GUI_1.py文件，点击一次”发送“按钮即显示一条消息

## text格式

每行为一条消息，每条消息必须以”Human：“或”AI：“开头，注意这里的冒号为中文格式，用以识别消息角色。
